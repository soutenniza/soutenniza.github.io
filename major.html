My Major - Computer Science


Important CS links

Wikipedia
This is the wikipedia link about computer science. This website tells you information about computer science. It also tells you how computer science can be split to many parts.
Codeacademy.com
Codeacademy is a good website to learn about coding. You can learn major languages like Ruby, PHP and Javascript.
Codeschool.com
Code School is also another great website for learning languages. Code school has videos and hands on exercises for Ruby, Javascript, HTML/CSS, and iOS
Cplusplus.com
CPlusPlus is an excellent website for your C++ needs. If you need to see if C++ has library you need. You can find it there and it tells you how to use all the functions too.
RailsforZombies.org
Rails for Zombies is a great website for people who want to learn how to code in ruby in rails. It includes video tutorials and hands on exercises on how to create your own twitter website using Ruby on rails.
